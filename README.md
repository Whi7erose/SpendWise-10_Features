# SpendWise 
# Google Sites link - https://sites.google.com/iit.ac.lk/spendwiselk/home

Prototype concept webpage for SpendWise — a personal budgeting app idea. Built for fun as a pet project.

## Pages

- `index.html` — Features hub (landing)
- `user-manual.html` — User manual
- `alerts.html` — Feature 01: Real-Time Pre-Purchase Alert
- `dashboards.html` — Feature 02: AI Spending Persona Dashboard
- `sms-tracking.html` — Feature 03: Bank SMS Auto-Parsing Feed
- `savings-map.html` — Feature 04: Geolocation Savings Map
- `voice-logging.html` — Feature 05: Voice Expense Logger
- `receipt-scan.html` — Feature 06: OCR Tax Receipt Scanner
- `reports.html` — Feature 07: SME Expense Report Generator
- `reimbursements.html` — Feature 08: Corporate Reimbursement Tracker
- `subscriptions.html` — Feature 09: Subscription Waste Detector
- `retail-insights.html` — Feature 10: B2B Retail Intelligence Dashboard

## How to run

No setup needed. Just open `index.html` in your browser.

## Tech

Plain HTML, CSS, and vanilla JavaScript. No frameworks, no build tools.

## Status

🚧 Prototype / concept only — not a real product (yet 👀)
